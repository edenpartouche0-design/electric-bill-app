# Smart Electric Bill App

אפליקציית Web עצמאית לחישוב חשבון חשמל לדיירים, עם העלאת PDF, חילוץ נתונים אוטומטי ואפשרות עריכה ידנית.

## יכולות עיקריות

- העלאת חשבונית חשמל בפורמט PDF
- חילוץ אוטומטי של:
  - תעריף לקוט"ש ללא מע"מ
  - תשלום קבוע
  - תשלום בגין הספק
  - אחוז מע"מ
- עריכה ידנית של כל ערך שחולץ
- הזנת קריאת מונה נוכחית וקריאת מונה קודמת
- חישוב חיוב לדייר עם דיוק של 2 ספרות אחרי הנקודה
- כרטיס סיכום ברור עם:
  - סך הקוט"ש שנצרך
  - עלות צריכה
  - חלק יחסי בתשלומים קבועים
  - מע"מ
  - סך הכול לתשלום

## מבנה הפרויקט

- `index.html` - נקודת הכניסה
- `styles.css` - עיצוב הממשק
- `script.js` - לוגיקת React, חילוץ PDF והחישובים

## איך מריצים

פותחים את `index.html` בדפדפן.

## הערות טכניות

- האפליקציה משתמשת ב־React דרך CDN
- חילוץ ה־PDF מבוסס על `pdf.js`
- כרגע אין שרת צד-שרת, וכל העבודה מתבצעת בדפדפן
